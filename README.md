# 📚 Guia de Estudos de Python com NotebookLM

## 📖 Sobre o projeto

Este projeto foi desenvolvido como parte do desafio da DIO **"Treinando uma IA de Aprendizagem: Explore o Poder do NotebookLM"**.

O objetivo é criar um guia de estudos sobre a linguagem Python utilizando o NotebookLM como ferramenta de apoio para organização do conhecimento, geração de resumos e revisão dos principais conceitos.

---

## 🎯 Objetivos

- Aprender os principais conceitos da linguagem Python.
- Organizar os estudos em um único material.
- Utilizar o NotebookLM para resumir conteúdos e esclarecer dúvidas.
- Desenvolver habilidades de pesquisa e documentação técnica.


---

## 📚 Curadoria de Fontes

As seguintes fontes foram selecionadas para compor o guia de estudos no NotebookLM:

1. Documentação Oficial do Python – https://docs.python.org/3/
2. Curso em Vídeo – Python (Gustavo Guanabara) – https://www.cursoemvideo.com/
3. W3Schools Python – https://www.w3schools.com/python/
4. Real Python – https://realpython.com/

Essas fontes foram escolhidas por serem confiáveis, atualizadas e amplamente utilizadas por estudantes e profissionais da área.

## 🤖 Engenharia de Prompts

Durante o desenvolvimento deste guia, foram utilizados prompts para explorar os recursos do NotebookLM.

### Exemplos de prompts utilizados

- Faça um resumo dos principais conceitos da linguagem Python.
- Explique a diferença entre listas, tuplas e dicionários.
- Quais são as estruturas de decisão em Python?

### Aprendizados

Os prompts permitiram organizar melhor os estudos, obter resumos mais objetivos e esclarecer dúvidas específicas sobre a linguagem Python.

# 📖 Miniguia de Estudos

## Resumo

Python é uma linguagem de programação de alto nível, conhecida por sua sintaxe simples e legível. É amplamente utilizada no desenvolvimento web, análise de dados, inteligência artificial, automação, computação científica e desenvolvimento de aplicações.

## Glossário

- **Variável:** espaço utilizado para armazenar um valor.
- **Função:** bloco de código reutilizável que executa uma tarefa.
- **Lista:** coleção ordenada e mutável de elementos.
- **Tupla:** coleção ordenada e imutável.
- **Dicionário:** coleção de pares chave e valor.
- **Loop:** estrutura utilizada para repetir instruções.
- **Condicional:** estrutura que permite tomar decisões utilizando `if`, `elif` e `else`.

## Prompts reutilizáveis

- Explique este conceito como se eu fosse iniciante.
- Crie um resumo em tópicos sobre este conteúdo.
- Gere cinco exercícios práticos sobre Python.
- Explique este código linha por linha.
- Quais são os erros mais comuns nesse assunto?

## Conclusão

O NotebookLM mostrou-se uma excelente ferramenta de apoio aos estudos, permitindo organizar conteúdos, gerar resumos, criar materiais de revisão e facilitar o aprendizado da linguagem Python.
